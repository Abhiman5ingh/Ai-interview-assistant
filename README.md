# AI Interview Assistant 🚀

An AI-powered platform that generates structured interview preparation reports using LLMs.

---

## 🧠 Powered By

* Groq SDK (LLM API)
* Node.js (Backend)
* React (Frontend)
* Zod (Schema Validation)

---

## ✨ Features

* 📊 Candidate match score
* 💡 Technical & behavioral questions
* 🧠 Answer strategies with interviewer intent
* 🔍 Skill gap analysis
* ⚡ Structured JSON output

## ⚙️ Setup

### 1. Clone the repo

```
git clone https://github.com/Abhiman5ingh/Ai-interview-assistant.git
```

### 2. Install dependencies

```
cd Backend && npm install
cd ../Frontend && npm install
```

### 3. Add Environment Variables

Create `.env` in Backend:

```
MONGODB_URL=your_mongo_url
JWT_SECRET= secret_jet_key
GROQ_API_KEY=your_api_key
```

### 4. Run the project

```
cd Backend && npm start
cd ../Frontend && npm run dev
```
